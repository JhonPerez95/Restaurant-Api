# Restaurant-Api
Manejo de inventario y de pedidos de un restaurante , CRUD

Manejo de tecnologias: 
- Nodejs
- Express
- Mongoose

En desarrollo:
- Dotenv
- Nodemon

Iniciar:
$ npm install

Ejecutar:
$ node src/index.js
